# Web Dashboard / Web 控制台

**[EN]** This is a React + Vite based interactive dashboard for managing Antigravity accounts, monitoring logs, and testing APIs. It provides a minimal setup to get React working in Vite with HMR and some ESLint rules.
**[中]** 这是一个基于 React + Vite 的交互式控制台，用于管理 Antigravity 账号、监控日志以及测试 API。它提供了一个最小化设置，支持 Vite 热更新（HMR）和 ESLint 规则。

---

## Technical Stack / 技术栈

**[EN]**
- **Framework**: React
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Plugins**: 
  - [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) for Fast Refresh.
  - [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) (Optional) uses SWC for even faster refresh.

**[中]**
- **框架**: React
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **插件**: 
  - [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) 支持快速刷新。
  - [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) （可选）使用 SWC 实现更快的刷新。

---

## Recommended Configuration / 推荐配置

**[EN]** For production applications, we recommend using TypeScript with type-aware lint rules. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io).
**[中]** 对于生产环境应用，我们建议使用 TypeScript 并启用类型感知的 Lint 规则。请参考 [TS 模板](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) 了解如何集成 TypeScript 和 [`typescript-eslint`](https://typescript-eslint.io)。
